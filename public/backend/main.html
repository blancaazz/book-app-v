  • Describe synthetically why and how did you manage session state, what are the state change triggering actions (e.g., POST to login etc..).

  1. /user/login and we send the user and password.
  2. If the data is correct with the database we set: a)req.session.loggedin = true b)req.session.name = user id
  3. Now the user logged in, and this will continue throw every page.
  4. For logging out we send a POST to /user/logout and now: a)req.session.loggedin = null b)req.session.name = null
